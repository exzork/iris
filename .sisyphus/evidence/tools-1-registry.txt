## Implementation Summary

### registry.go Changes
Added one new method to *Registry:

1. OpenAIFunctions() []map[string]interface{}
   - Returns all non-admin tools in OpenAI function-calling format
   - Acquires read lock on registry mutex
   - Iterates all registered tool definitions
   - Skips tools with AdminOnly=true
   - Calls schema.ToOpenAIFunction() for each non-admin tool
   - Returns empty slice (not nil) when no tools registered or all are admin
   - Thread-safe with RWMutex protection

### registry_test.go Changes
Added two new tests:

1. TestRegistry_OpenAIFunctions_SkipsAdminTools
   - Registers one normal tool (fakeEchoTool, AdminOnly=false)
   - Registers one admin tool (fakeAdminTool, AdminOnly=true)
   - Calls OpenAIFunctions()
   - Asserts result length is 1 (only normal tool)
   - Verifies returned function name is "echo_tool"
   - Confirms admin tool is excluded

2. TestRegistry_OpenAIFunctions_RoundTrip
   - Registers one normal tool (fakeEchoTool)
   - Calls OpenAIFunctions()
   - Marshals result[0] to JSON using encoding/json.Marshal
   - Unmarshals back to map[string]interface{}
   - Verifies structure preserved: type, function, name, parameters
   - Confirms JSON round-trip maintains exact shape
