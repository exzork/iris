## Key Behavior Changes

1. **Timeout Semantics**
   - 0 = no deadline (only parent ctx cancellation)
   - > 0 = deadline with specified duration
   - < 0 = invalid (substituted with default)

2. **Model Selection**
   - All lore LLM calls use IRIS_LORE_LLM_MODEL
   - Defaults to LLM_MODEL_STRONG when unset
   - Single source of truth for lore model configuration

3. **Graceful Shutdown**
   - Bot respects SIGTERM via root context cancellation
   - Lore LLM calls interrupted on root ctx cancel
   - No premature timeouts with IRIS_LORE_LLM_TIMEOUT=0
