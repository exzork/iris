## Implementation Details

### Delta Buffering Pattern
- OnDelta callback fires for each text fragment
- Tool-call arguments accumulated in map[int]*pendingToolCall keyed by index
- Arguments assembled via strings.Builder before JSON parsing

### Tool-Call Accumulation
- Index-based keying handles parallel tool calls
- ID and name fields accumulated across deltas (if fragment.ID != "" { p.id = fragment.ID })
- Arguments concatenated as JSON fragments arrive

### Error Handling
- Invalid JSON arguments: tool response content = "error: invalid JSON arguments"
- Tool execution errors: tool response content = "error: <exec error>"
- Stream errors: propagated with context

### Multi-Round Loop
- Continues until finish_reason="stop" or max rounds exceeded
- Tool results appended to message history per OpenAI protocol
- Preserves ChatWithTools (non-streaming) for classifiers
